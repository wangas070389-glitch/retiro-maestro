# Análisis Red Team 017: Vulneración de Inputs Certificados

**Fecha:** 24/02/2026
**Objetivo:** `scenarioA` State Management & Form Inputs
**Vulnerabilidad Evaluada:** Mutación silenciosa de datos validados forensemente.

## 1. El Vector de Ataque
1. El asesor carga un PDF oficial válido (`thelma.pdf`). El `DocumentUploader` extrae los datos correctamente.
2. El sistema sella la transacción con un 100% de confianza.
3. El asesor, por error o malicia, decide incrementar "Semanas Cotizadas" de 1135 a 1500 en el input box estándar de la calculadora.
4. Si el sistema no invalida el sello, el motor generará un cálculo con las 1500 semanas dictadas por el asesor, **PERO** el reporte final en PDF imprimirá una advertencia de "Extracción Verificada al 100% desde thelma.pdf". 

Esto permite falsificar proyecciones oficiales, rompiendo por completo la confianza en el acta.

## 2. Contramedida Implementada (Friction Destructor)
La única defensa robusta es asumir que el estado de validación (`certified_dossier`) es frágil como el cristal.

* **Trigger:** Cualquier evento `onChange` o dispatch proveniente de los inputs humanos (Semanas, Salario, Edad) debe ser interceptado en el dashboard.
* **Payload:** Si se detecta un cambio manual, el bloque UI debe despachar un `setCertifiedDossier(null)`, destruyendo instantáneamente el sello de confiabilidad, y regresando la simulación a un modo "Inercial Especulativo" sin marcas de agua de Certificación oficial.
* **UX Tradeoff:** Se optó por permitir la mutación disparando una alerta ("Precisión Forense Rota"), en lugar de deshabilitar (disable) los inputs por completo, ya que el operador podría estar haciendo un ejercicio "What-if" legítimo basado en el PDF y no debe verse restringido algorítmicamente de proyectar el futuro.

## 3. Puntuaciones CVSS V3 (Estimadas Previas a Contramedida)
**Vector:** Local (Re-input UI)
**Impacto de Integridad:** High (Falsificación de proyecciones oficiales bajo marca IMSS)
**Severidad Base:** 7.5 (High)

*Con la contramedida mitigada a 0.0 (Info).*
