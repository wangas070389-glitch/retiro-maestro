# N2-019: Ingestion Verification Interaction Flow

## 1. Interaction Diagram

This sequence details the interaction between the User UI, the Local Extractors, and the State Engine during document ingestion.

```mermaid
sequenceDiagram
    actor User
    participant Browser UI (Dropzone)
    participant PDF Worker
    participant HeuristicParser
    participant SentinelAuditor
    participant Zustand Store

    User->>Browser UI (Dropzone): Drops "Constancia.pdf"
    Browser UI (Dropzone)->>Browser UI (Dropzone): Validates Size & MIME (<5MB)
    Browser UI (Dropzone)->>Browser UI (Dropzone): State: PROCESSING
    Browser UI (Dropzone)->>PDF Worker: Send Binary Data
    PDF Worker-->>Browser UI (Dropzone): Return Raw Extracted Text
    Browser UI (Dropzone)->>HeuristicParser: parse(Raw Text)
    HeuristicParser-->>Browser UI (Dropzone): Return IngestedData (Weeks, Salary, NSS...)
    Browser UI (Dropzone)->>SentinelAuditor: audit(IngestedData)
    SentinelAuditor-->>Browser UI (Dropzone): Return AuditReport (Valid/Invalid, Score, Flags)
    
    alt Audit is Valid (Score > 0.65)
        Browser UI (Dropzone)->>Browser UI (Dropzone): State: REVIEW (Green Theme)
        User->>Browser UI (Dropzone): Clicks "Confirm"
        Browser UI (Dropzone)->>Zustand Store: Dispatch Validated Payload
    else Audit is Invalid / Flagged
        Browser UI (Dropzone)->>Browser UI (Dropzone): State: REVIEW (Red/Amber Theme)
        Browser UI (Dropzone)->>User: Prompt manual correction on flagged fields
        User->>Browser UI (Dropzone): Edits fields manually
        User->>Browser UI (Dropzone): Clicks "Confirm Override"
        Browser UI (Dropzone)->>Zustand Store: Dispatch Overridden Payload
    end
```

## 2. Kinetic Feedback Loops
- **Processing Phase**: Fast, pulsing gradient animation to indicate local computation.
- **Review Phase (Valid)**: Staggered reveal of pre-filled inputs with green checkmarks.
- **Review Phase (Flagged)**: Targeted red borders on specific inputs (e.g., DOB) with an inline rendering of the Sentinel recommendation text.
