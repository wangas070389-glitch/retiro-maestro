# N2-015: Deep Analysis Flow

## User Journey: "The Skeptic"
*User wants to know if paying 5 years of Modalidad 40 is actually worth it.*

```mermaid
sequenceDiagram
    participant User
    participant Dashboard
    participant StrategyCards
    participant Modal
    participant Engine

    User->>Dashboard: Enters Data (1250 weeks, $500 salary)
    Dashboard->>StrategyCards: Renders "Topado" (ROI 13 months)
    
    User->>StrategyCards: Clicks "Ver Detalle Topado"
    StrategyCards->>Modal: Open(StrategyID: "topado")
    Modal->>Engine: calculateProjection("topado")
    Engine-->>Modal: Returns Array[5] (Years 1-5)
    
    Modal->>User: Shows "Pension Growth" Chart
    note right of Modal: Initial view is the Graph. <br/> Shows steep curve of pension growth.
    
    User->>Modal: Clicks "Tabla" Tab
    Modal->>User: Shows numbers Grid
    
    User->>Modal: Clicks "Cerrar"
    Modal->>Dashboard: Return focus to Card
```

## Micro-interactions
-   **Chart Hover**: When hovering year 3 on the chart, show a tooltip with "Pensión: $45,000 | Inversión: $300,000".
-   **Tab Switch**: Smooth `framer-motion` slide between Chart and Table views.
