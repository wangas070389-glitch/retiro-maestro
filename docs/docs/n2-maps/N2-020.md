# N2-020: Flujo de Ingestión Forense y Certificación Criptológica

Este diagrama ilustra el flujo Criptológico de Procedencia de Datos, desde que el Archivo Físico toca el Cliente, hasta que se emite su huella en el Acta Final y se registra en el Ledger Subyacente, obedeciendo a la arquitectura STRAT-025.

```mermaid
sequenceDiagram
    participant Operador
    participant DocumentUploader
    participant SentinelAuditor
    participant ZustandStore
    participant DashboardUI
    participant PDFGenerator
    participant LedgerEngine

    %% Fase 1: Extracción Física a Criptología Local
    Operador->>DocumentUploader: Drop PDF Oficial Constancia IMSS
    DocumentUploader->>SentinelAuditor: Envía texto plano (No-Red, 100% Local)
    SentinelAuditor-->>DocumentUploader: Retorna {AuditReport, Confidence: 100%}
    
    note over DocumentUploader,ZustandStore: Minting (Acuñado) de Estado
    DocumentUploader->>ZustandStore: setCertifiedDossier(filaname, nss, curp)
    DocumentUploader->>ZustandStore: updateScenarioA(semanas, salario, edad)
    
    %% Fase 2: Presentación Reactiva y Protección
    ZustandStore-->>DashboardUI: Propaga Datos & Certificado
    DashboardUI-->>Operador: Muestra "Soberanía Comprobada" (Candados Activos)
    
    %% Escenario de Fricción Evaluado en RED-017
    alt Mutación Manual de Input
        Operador->>DashboardUI: Cambia Salary Promedio a Mano
        DashboardUI->>ZustandStore: Destruye Certificado: setCertifiedDossier(null)
        ZustandStore-->>DashboardUI: Remueve "Soberanía Comprobada"
    end
    
    %% Fase 3: Renderizado Final Indisoluble
    Operador->>DashboardUI: Click "Calcular Pensión"
    DashboardUI->>DashboardUI: Genera Cálculo Local PensionEngine()
    DashboardUI->>PDFGenerator: Solicita Render (Data + Certificado Si Existe)
    
    alt Sello Forense Intacto
        PDFGenerator-->>Operador: Retorna PDF Acta_Inercial_Soberana.pdf con Marca de Agua de Autenticidad Extractora
    else Sello Destruido (Simulación Especulativa)
        PDFGenerator-->>Operador: Retorna PDF Limpio, Declarando "Naturaleza Especulativa" sin Firmas
    end
    
    %% Guardado
    Operador->>DashboardUI: Guardar en Expediente
    DashboardUI->>LedgerEngine: Guarda Resultado de Estrategia INERCIAL (Con o sin Flag de Origen)
```
