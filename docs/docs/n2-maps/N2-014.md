# N2-014: Interaction Map (Kinetic & Mobile)

## 1. Navigation Flow (Mobile)
Unlike the Desktop "Command Center", the Mobile flow focuses on focused tasks.

```mermaid
graph TD
    A[Mobile Entry] --> B{Menu Button}
    B -->|Tap| C[Sidebar Sheet (Slide In)]
    C -->|Select| D[Module View]
    D -->|Swipe Right| C
    D -->|Scroll Down| E[Hide Header]
    D -->|Scroll Up| F[Show Header]
```

## 2. Micro-Interactions
Defining the "Feel" of every click.

### A. Strategy Cards
-   **State: Idle**: Glass background, 0% elevation.
-   **State: Hover (Desktop)**: Scale 1.02x, Inner Glow opacity 100%.
-   **State: Tap (Mobile)**: Scale 0.98x (Tactile feedback).
-   **Action: Delete**: Swipe Left (Mobile) / Fade Out + Layout Shift (Desktop).

### B. Input Fields (Laboratory)
-   **State: Focus**: Border color transitions to `indigo-500`, Label floats up (if applicable).
-   **State: Error**: Shake animation (X-axis), border `red-500`.

### C. Simulation Results
-   **Trigger**: User updates `Salary` or `Weeks`.
-   **Response**:
    1.  Show linear progress bar (skeleton) for < 200ms.
    2.  Result appears.
    3.  Numbers "tick up" from old value to new value over 500ms.

## 3. Gestures
*   **Swipe Left (Card)**: Reveal "Delete" or "Archive" action.
*   **Pull to Refresh**: Reload Simulation Data.
*   **Long Press (Strategy)**: Enter "Compare Mode".
