# Compact High-Density Dashboard Data Flow Map (N2-025)

## Flow Overview
This diagram models the responsive data flow, UI layout nodes, and state updates that occur when inputs are manipulated, simulations are loaded, or payments are toggled.

```mermaid
graph TD
    %% User/Advisor Inputs
    Inputs[User & Advisor Inputs] -->|mutate| StoreState[Simulation Store: State]
    StoreState -->|re-render| FormLayout[Left Column: Input Fields Grid]
    
    %% Forms and Actions
    FormLayout -->|Submit| CalcAction[calculatePensionAction]
    CalcAction -->|Validate| ZodSchema[Zod: PensionSchema]
    ZodSchema -->|Defaults| RunEngine[PensionEngine: calculate]
    
    %% Engine to outputs
    RunEngine -->|Output| UIResult[Right Column: Results Panel]
    UIResult -->|Render| NetHero[Net Pension Hero Block]
    UIResult -->|Render| CompGrid[2x2 Composition Grid]
    UIResult -->|Action| SaveBtn[Guardar Estudio button]
    UIResult -->|Action| PDFBtns[Brief & Comprehensive PDF exports]
    
    %% Saved simulation history
    SaveBtn -->|saveSimulationAction| DB_Sims[(Database: Simulations)]
    DB_Sims -->|Load List| SavedSimList[Mis Estudios Table]
    SavedSimList -->|Select Study| StoreState
    
    %% Layout Constraints (No Scroll Mode)
    style FormLayout fill:#f3f4f6,stroke:#d1d5db,stroke-width:1px
    style UIResult fill:#f3f4f6,stroke:#d1d5db,stroke-width:1px
    style SavedSimList fill:#eff6ff,stroke:#bfdbfe,stroke-width:2px
```

## Layout Grid Node Specifications
1. **Left Column (`lg:col-span-5`)**:
   - `GridRow 1`: Age & Retirement Age inputs (`h-[38px]`).
   - `GridRow 2`: Weeks & Average Salary inputs (`h-[38px]`).
   - `GridRow 3`: Date of Baja (`h-[38px]`) & Estatus Laboral Checkbox card (`h-[38px] px-3`).
   - `Details`: Collapsible accordions for Opciones Avanzadas and Asignaciones Familiares.
2. **Right Column (`lg:col-span-7`)**:
   - `Hero Card`: Display net estimated pension monthly amount.
   - `Breakdown`: Compact 2x2 grid metrics cards.
   - `Actions`: Symmetric `grid-cols-3` row for Save, Brief PDF, and Comprehensive PDF.
   - `History Table`: Fixed container with `max-h-[160px] overflow-y-auto` and `sticky top-0` table header.
