# 📜 ADR: Adoption of GenericContext-Geminis Framework

## Context
The Retiro Maestro project requires a robust, sovereign, and audit-ready technical operating system to manage high-stakes financial calculations (IMSS Ley 73 Pension). To ensure complexity management and adversarial validation, a structured framework is needed as detected by the initial assessment of the "Architect-Engineer 2026" vision.

## Options Considered
* **Option A**: Ad-hoc documentation and tribal knowledge. (Rejected due to high regulatory/financial risk).
* **Option B**: Standard enterprise documentation (WIKI). (Rejected due to lack of sovereign-agentic alignment).
* **Option C**: **GenericContext-Geminis Framework**. (Selected for its focus on precision, adversarial testing, and documentation as a "Machine Constitution").

## Decision
Adopt the **GenericContext-Geminis** framework as the "Constitutional Foundation" for Retiro Maestro, prioritizing **Precision** and **Auditability** as the North Star attributes.

## Consequences
* **Positive**: Enhanced reliability of pension calculations, clear accountability (Sovereignty), and systematic risk reduction via Red Teaming.
* **Neutral**: Steep learning curve for the "Architect-Engineer" persona.
* **Negative**: Increased documentation overhead for minor tactical changes.
