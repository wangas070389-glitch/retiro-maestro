# ADR-035: Free Trial Sovereignty & Limits

## Status
**ACTIVE**

## Context
A free trial system is required to allow initial engagement while preventing unlimited use by free-tier accounts. The system needs to be structurally decoupled from future payment gateways but enforce strict actuarial usage limits.

## Decision
1. **Time Domain Constraint**: 7 calendar days starting directly from `User.createdAt`. This avoids needing a separate timestamp column.
2. **Engagement Constraint**: Maximum of 2 saved studies, tracked via `trialSimulationsUsed` in the Prisma schema.
3. **Triple-Ring Enforcement**:
   - **Visual Nudge**: `TrialBadge` indicating days/studies remaining.
   - **UI Lockout**: `TrialLockout` full-screen overlay when constraints are breached.
   - **Cryptographic Gate**: `assertTrialAccess()` server-side barrier on both calculation and persistence layers.
4. **Sovereignty Exemption**: Only `ADMIN` roles inherently bypass all trial restrictions. `ADVISOR` roles are now strictly subject to the same operational trial constraints as users.
