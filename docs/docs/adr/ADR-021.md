# ADR-021: Sovereign Unification (Dashboard Consolidation)

## Status
Proposed (Implemented)

## Context
The "Laboratory" screen was initially designed as a sandbox for experimental tools (ISR Tax, Retroactive Calculator, Loan Simulator). However, this created a fragmented user experience where forensic planning happened in silos. To achieve a "Sovereign Workflow," all tools must be accessible from the unified command center (Dashboard).

## Decision
We decided to decommission the standalone `Laboratory` page and integrate its core functionality into the main `Dashboard`:

1.  **ISR Calculation**: Integrated into the `PensionEngine` and displayed alongside Gross Pension in the Dashboard results.
2.  **Extended Family Logic**: Fields for "Hijos" and "Padres" are now permanent members of the Dashboard input form.
3.  **Optimization Lab**: Added a dedicated "Laboratorio de Optimización" section in the Dashboard UI to house the Retroactive and Loan simulators.

## Consequences
- **Positive**: Simplified navigation, unified state (injecting retroactive weeks updates calculation results immediately), more professional UI.
- **Negative**: Increased complexity of the Dashboard component (mitigated by better sectioning).
- **Forensic**: PDF reports now consistently account for children and parents without needing manual overrides.
